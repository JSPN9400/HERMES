# HERMES v0.1

HERMES (Hierarchical Expert Routing via Semantic Graphs) is an open research hypothesis.

Status: Idea stage (not validated).

Goal:
Explore whether semantic graph routing, task decomposition and expert coordination can reduce unnecessary global reasoning while preserving answer quality.

Roadmap:
1. Literature Review
2. Architecture
3. Mathematical Model
4. Routing Algorithm
5. Complexity Analysis
6. Python Prototype
7. Benchmarks
8. Research Paper
